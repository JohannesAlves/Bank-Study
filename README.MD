## Instalação

## Server App

1. Na pasta SERVER, execute o comando:

```bash
  npm i
  npx prisma db pull
  npx prisma migrate dev
```

2. Execute o SERVER:

```bash
  npm run dev
```

## Web App

1. Na pasta WEB, execute o comando:

```bash
  npm i
  npm run dev
```
